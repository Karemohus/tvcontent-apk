# TV Content APK

نظام Digital Signage احترافي للشاشات التجارية.

## 📦 المميزات

- ✅ **Auto-Start** عند تشغيل الجهاز
- ✅ **Fullscreen Kiosk Mode** بدون أزرار
- ✅ **Offline Caching** يعمل بدون إنترنت
- ✅ **Auto-Recovery** عند انقطاع الاتصال
- ✅ **WebView** يدعم كل ميزات الموقع
- ✅ **Bilingual** عربي / إنجليزي

## 🛠️ الإعدادات

- اضغط على الشعار **10 مرات** للوصول للإعدادات
- الرمز السري الافتراضي: `9999`
- عنوان الخادم الافتراضي: `https://aljabr.duckdns.org/`

## 📲 التثبيت على TV Box

1. فعّل "Unknown sources" في إعدادات الجهاز
2. نزّل `TVContent-v1.0.0.apk` من قسم Releases
3. ثبّت التطبيق
4. افتح التطبيق — يظهر كود من 6 أرقام
5. سجّل الكود من Dashboard

## 🏗️ البناء

البناء يحصل تلقائياً عبر GitHub Actions عند كل `push`.

النتيجة: APK جاهز في قسم Releases.
